## Search & Show File

|**Command**|**Description**|
|---|---|
|**SEARCH**||
|`SearchFileName "Name"`|Search for part of filename|
|`SearchFolderName "Name"`|Search for part of foldername|
|`SearchContent "Hi"`|Search inside files with RipGrep|
|**SHOW FILE**||
|`ShowCsvInGridView "File.csv"`|Show CSV in GridView|
|`ShowCsvInGridView "File.csv" ";"`|GridView with custom delimiter|
|`ShowCsvInGridView "File.csv" "t"`|GridView with custom delimiter|
|`ShowJsonInGridView "File.json"`|Show JSON array in GridView|
|`ShowJsonInGridView "File.json" "data"`|Provide a key for the array|
